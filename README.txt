資金繰りメーター 試作版

内容
- index.html: アプリ本体
- styles.css / app.js: 画面と計算処理
- manifest.webmanifest / service-worker.js: ホーム画面追加・オフライン利用用

主な機能
- 個人口座、バンド口座、金庫、FCプール入力
- 入金・支払い項目の追加削除
- 支払い後残高の自動計算
- 残高に応じた「赤字／危険／警戒／安定」判定
- 分割シミュレーション
- 端末内保存

スマホで試す方法
1. フォルダ一式をWebサーバーへ置く
2. SafariまたはChromeでURLを開く
3. iPhone: 共有→ホーム画面に追加
   Android: メニュー→アプリをインストール／ホーム画面に追加

注意
- 銀行やカード会社とは連携しません
- データは利用端末のブラウザ内だけに保存されます
